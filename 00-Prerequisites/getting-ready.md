## Prerequisites Self-Check

Before starting the course, make sure you're comfortable with these foundational skills:

### ‚úÖ Programming Basics

You should be able to:
- [ ] Create and use variables
- [ ] Write and call functions
- [ ] Use if/else statements
- [ ] Work with loops (for, while)
- [ ] Understand basic data types (strings, numbers, booleans, lists/arrays)

**Need to learn?** 
- Python: [Python.org Tutorial](https://docs.python.org/3/tutorial/)
- JavaScript: [MDN JavaScript Guide](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide)

### ‚úÖ JSON Basics

You should know:
- [ ] What JSON is (JavaScript Object Notation)
- [ ] How to read JSON data
- [ ] Basic JSON structure (objects, arrays, key-value pairs)

**Example JSON:**
```json
{
  "name": "Alice",
  "age": 25,
  "skills": ["Python", "JavaScript"],
  "active": true
}
```

**Need to learn?**
- [JSON Introduction](https://www.json.org/)
- [Working with JSON - MDN](https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Objects/JSON)

### ‚úÖ Command Line Usage

You should be able to:
- [ ] Open a terminal/command prompt
- [ ] Navigate directories (`cd`)
- [ ] List files (`ls` on Mac/Linux, `dir` on Windows)
- [ ] Run programs from the command line
- [ ] Understand what a path is

**Basic Commands:**
```bash
# Navigate to a directory
cd Documents

# Go back one directory
cd ..

# List files in current directory
ls        # Mac/Linux
dir       # Windows

# Show current directory
pwd       # Mac/Linux
cd        # Windows

# Run a Python script
python my_script.py
```

**Need to learn?**
- [Command Line Crash Course](https://www.freecodecamp.org/news/command-line-for-beginners/)
- [Linux/Mac Terminal Basics](https://ubuntu.com/tutorials/command-line-for-beginners)

### ‚úÖ Software Installation

You should be comfortable:
- [ ] Downloading and installing software
- [ ] Following installation instructions
- [ ] Basic troubleshooting (searching for error messages)

## Required Software

Before starting the course, please install:

### 1. Python 3.7 or higher

**Check if installed:**
```bash
python --version
# or
python3 --version
```

**Installation:**
- **Windows**: [Download from Python.org](https://www.python.org/downloads/)
- **Mac**: Pre-installed, or use `brew install python3`
- **Linux**: Usually pre-installed, or use `sudo apt install python3`

### 2. Text Editor or IDE

Choose one:
- **VS Code** (Recommended) - [Download here](https://code.visualstudio.com/)
- **PyCharm Community** - [Download here](https://www.jetbrains.com/pycharm/download/)
- **Sublime Text** - [Download here](https://www.sublimetext.com/)
- **Any text editor you're comfortable with**

### 3. Git (for downloading course materials)

**Check if installed:**
```bash
git --version
```

**Installation:**
- **Windows**: [Download Git for Windows](https://git-scm.com/download/win)
- **Mac**: `brew install git` or [Download here](https://git-scm.com/download/mac)
- **Linux**: `sudo apt install git`

## Quick Skill Builders

If you need to strengthen your foundation before starting:

### Python Quick Start (1-2 days)
1. [Python Beginner Tutorial](https://www.learnpython.org/) - Interactive lessons
2. Write a simple calculator program
3. Create a program that reads/writes JSON files

### JavaScript Quick Start (1-2 days)
1. [JavaScript.info Basics](https://javascript.info/first-steps) - Clear explanations
2. Write functions that manipulate data
3. Practice with JSON.parse() and JSON.stringify()

### Command Line Basics (2-3 hours)
1. Practice navigating your file system
2. Create, move, and delete files/folders
3. Learn to read error messages

## Books

### Beginner-Friendly

**Computer Networking: A Top-Down Approach** by Kurose & Ross
- Excellent for understanding networking concepts
- Starts with application layer (HTTP, DNS)
- Clear explanations with real-world examples

**HTTP: The Definitive Guide** by Gourley & Totty
- Deep dive into HTTP protocol
- Great reference for web developers
- Covers HTTP/1.1 in detail

**RESTful Web APIs** by Leonard Richardson & Mike Amundsen
- Comprehensive guide to REST API design
- Practical examples
- Best practices and patterns

### Intermediate

**TCP/IP Illustrated, Volume 1** by W. Richard Stevens
- The classic networking book
- Technical but readable
- Deep understanding of TCP/IP

**Designing Data-Intensive Applications** by Martin Kleppmann
- Modern distributed systems
- Covers networking, databases, caching
- Real-world system design

**Web Security for Developers** by Malcolm McDonald
- Practical security guide
- Covers HTTPS, authentication, CORS
- Easy to follow examples

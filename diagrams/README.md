Each diagram is referenced in the corresponding module's `diagrams.md` file. The diagrams use:
- ASCII art for text-based representations

- Descriptions for complex architectures

## Contributing

When adding diagrams:
1. Keep them simple and focused on one concept
2. Use consistent symbols and notation
3. Include a brief description of what the diagram illustrates
4. Reference the diagram in the relevant module's content

# Module 00 - Prerequisites Assessment

## Self-Graded Quiz

### Section 1: Environment Setup

**Question 1:** What command creates a Python virtual environment?
```
a) python install venv
b) python3 -m venv venv
c) pip create venv
d) virtualenv --new
```
<details><summary>Answer</summary>**b)** `python3 -m venv venv`</details>

**Question 2:** How do you activate a virtual environment on Linux/Mac?
```
a) activate venv
b) venv/start
c) source venv/bin/activate
d) run venv
```
<details><summary>Answer</summary>**c)** `source venv/bin/activate`</details>

**Question 3:** What does `pip install -r requirements.txt` do?
```
a) Creates a requirements file
b) Installs all packages listed in requirements.txt
c) Removes all packages
d) Updates pip
```
<details><summary>Answer</summary>**b)** Installs all packages listed in requirements.txt</details>

### Section 2: Command Line Basics

**Question 4:** Which command shows your current directory?
```
a) dir
b) ls
c) pwd
d) cd
```
<details><summary>Answer</summary>**c)** `pwd` (Print Working Directory)</details>

**Question 5:** How do you navigate to the parent directory?
```
a) cd parent
b) cd ..
c) cd back
d) cd up
```
<details><summary>Answer</summary>**b)** `cd ..`</details>

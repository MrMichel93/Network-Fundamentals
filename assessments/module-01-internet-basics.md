# Module 01 - How The Internet Works Assessment

## Self-Graded Quiz

### Section 1: IP Addresses and DNS

**Question 1:** How many octets are in an IPv4 address?
```
a) 2
b) 4
c) 6
d) 8
```
<details><summary>Answer</summary>**b)** 4 octets (e.g., 192.168.1.1)</details>

**Question 2:** What does DNS stand for?
```
a) Data Network System
b) Domain Name System
c) Digital Network Service
d) Dynamic Name Server
```
<details><summary>Answer</summary>**b)** Domain Name System</details>

**Question 3:** Which of these is a private IP address?
```
a) 8.8.8.8
b) 1.1.1.1
c) 192.168.1.100
d) 142.250.185.46
```
<details><summary>Answer</summary>**c)** 192.168.x.x addresses are private</details>

### Section 2: Network Tools

**Question 4:** What does the ping command test?
```
a) DNS resolution speed
b) Connection speed and reachability
c) Router configuration
d) Firewall settings
```
<details><summary>Answer</summary>**b)** Tests reachability and measures latency</details>

**Question 5:** What does traceroute show?
```
a) All websites visited
b) The path packets take to reach a destination
c) DNS cache contents
d) IP address history
```
<details><summary>Answer</summary>**b)** Shows each hop/router along the path</details>

This directory contains quizzes, checkpoints, and self-assessment materials for each module.

## Purpose

Help learners validate their understanding through:
- Module quizzes (multiple choice, true/false, short answer)
- Checkpoint assessments to gauge progress
- Self-reflection questions
- Practical scenario challenges

## Structure

Assessments are organized by module:
- `module-00-prerequisites.md`
- `module-01-internet-basics.md`
- `module-02-http-fundamentals.md`
- `module-03-rest-apis.md`
- `module-04-websockets.md`
- `module-05-other-protocols.md`
- `module-06-security-basics.md`

## How to Use

1. Complete each module's lessons before attempting the assessment
2. Try to answer questions without referring to the material first
3. Check your answers against the provided solutions
4. Review areas where you struggled before moving to the next module

## Grading

These assessments are self-graded for learning purposes. Aim for:
- 80%+ understanding before moving forward
- Ability to explain concepts in your own words
- Confidence to apply concepts in practical exercises

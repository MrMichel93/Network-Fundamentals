## üéØ Learning Path

### Beginners
1. Start with [Public APIs](./practice-apis.md)
2. Try APIs with no authentication (JSONPlaceholder, HTTPBin)
3. Use curl or Python to make requests
4. Complete the practice challenges

### Intermediate
1. Set up the [Local Practice API](./local-practice-api/)
2. Understand authentication with JWT
3. Practice all CRUD operations
4. Build a client application
5. Try APIs with API key authentication

### Advanced
1. Implement OAuth flow with GitHub/Twitter
2. Add features to the Local Practice API
3. Build a multi-API mashup project
4. Deploy your own API to production
